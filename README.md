scalabot
========

An IRC bot to autovoice trusted people

The simple config file would be:


        {
            "alkar" : {
                "host" : "irc.tsua.net",
                "port" : 9990,
                "user" : {
                    "login" : "username",
                    "name" : "cool name",
                    "password" : "password"
                },
                "voices" : ["user1", "user2"]
            }
        }
